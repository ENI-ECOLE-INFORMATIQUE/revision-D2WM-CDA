# 🚀 Révision D2WM → CDA

Bienvenue sur ce dépôt de révision dédié aux apprenants en formation  
**D2WM (Développeur Web & Web Mobile)** et **CDA (Concepteur Développeur d’Applications)**

🎯 Objectif : consolider les compétences, pratiquer et monter en niveau pour réussir le passage vers le CDA et le monde professionnel.

---

## 📚 Contenu du dépôt

Ce repository regroupe différents supports de révision :

- 🧠 **Rappels de cours essentiels**
- 💻 **Exercices pratiques**
- 🧪 **Katas & challenges**
- 🛠️ **Mini-projets**
- 📄 **Bonnes pratiques & méthodes**
- ❓ **Questions types examen / entretien**

---

## 🧩 Compétences travaillées

### 🔹 Front-end
- HTML / CSS
- JavaScript
- Responsive design
- Accessibilité

### 🔹 Back-end
- PHP / Node.js (selon stack)
- API REST
- Authentification / Sécurité

### 🔹 Base de données
- SQL (MySQL / PostgreSQL)
- Modélisation (MCD / MLD)

### 🔹 Outils & Méthodologies
- Git / GitHub
- Docker (initiation)
- Méthodes Agile
- Tests & qualité logicielle

---

## 🎓 Contexte pédagogique

Les formations D2WM et CDA permettent d’accéder à des compétences professionnelles reconnues dans le développement logiciel, avec des titres inscrits au RNCP.

👉 Ce dépôt vise à :
- renforcer les bases techniques
- préparer les projets professionnels
- améliorer l’autonomie des apprenants
- favoriser l’employabilité
